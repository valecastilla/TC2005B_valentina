CSS

¿Qué es css?

El CSS nos permite agregar diseño y estética a nuestras páginas web, además de compartir estilos entre diferentes elementos y páginas.

Selectores

Un selector indica qué elemento o elementos dentro de nuestro HTML se van a modificar para aplicarles estilos (como color, tamaño y posición). Los selectores pueden incluir una combinación de diferentes calificadores para seleccionar elementos únicos, dependiendo del nivel de especificidad.

Ej. p { ... }

Existen distintos tipos de selectores:

Selector de tipo Los selectores de tipo seleccionan elementos según su tipo de elemento.

Selector de clase Los selectores de clase seleccionan elementos según su clase, son más especificos que los de tipo. Se denotan con un punto (.) seguido del nombre de la clase.

Selector de id Los selectores de id seleccionan elementos según su id, son más especificos que los de clase y tipo. Se denotan con un numeral (#) seguido del nombre del id.

Propiedades

Una propiedad determina los estilos que se le aplicarán. Los nombres de las propiedades se escriben dentro del selector, algunos ejemplos son:

  • background
  • color
  • font-size
  • height
  • width
Ej. p { color: ...; font-size: ...; }

Valores

Un valor determina el comportamiento de una propiedad. Los valores se escriben siguiendo el nombre de la propiedad, algunos ejemplos son:

Ej. p { color: orange; font-size: 16px; }
