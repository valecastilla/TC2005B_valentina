Git y github

¿Qué es git?

Git es un software de control de versiones gratis de open source diseñado por Linus Torvalds. Permite registrar los cambios realizados a archivos de computadora (Ej. Código fuente). Git hace esto a través de repositorios (almacenamiento virtual de tu proyecto).

¿Qué es github?

Github es una plataforma web que permite almacenar los repositorios de manera remota usando el sistema de control de versiones git, con el fin de colaborar y coordinar los avances de un proyecto que pertenece a un equipo de trabajo. Un repositorio remoto permite a los equipos de desarrollo de software aplicar múltiples cambios a un proyecto. En lugar de aplicar los cambios directamente, utilizan ciertas características de un repositorio para editar y revisar los cambios antes de aplicarlos. Esto reduce los conflictos de cuando dos o más desarrolladores editan una misma parte del código.
